## Basic
 info -> Information Guide

 result -> Last Result Matrix

 add scaler -> Add Scaler
 mult scaler -> Multipliy Scaler
 det -> Determinant
 transpose -> Transpose
 minor -> Minor Matrix
 cofactor -> Cofactor Matrix
 adjoint -> Adjoint Matrix
 inverse -> Inverse Matrix
 pow -> Power Matrix

## Binary Transforms
 add -> Add Matrices (with scale)
 mult -> Multipliy Matrices (with scale)
